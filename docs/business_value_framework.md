# Business Value Framework

## Time Saved

The system reduces first-pass research time by turning an analyst question into a structured workflow:

```text
Question -> Scenario Family -> Analogues -> Market Evidence -> Pathways -> Memo
```

Analysts still review the evidence, but they start from a consistent structure rather than a blank page.

## Research Standardization

The repository standardizes:

- scenario families;
- question framing;
- historical analogue retrieval;
- market evidence documentation;
- pathway language;
- executive memo structure.

This reduces inconsistent outputs across analysts and use cases.

## Evidence Transparency

The workflow preserves visible evidence:

- classification terms;
- retrieved analogues;
- similarity rationale;
- event coding notes;
- market proxy choices;
- benchmark and return-window notes;
- limitations.

This helps reviewers understand how the output was produced.

## Historical Context Generation

Analysts often need fast context before discussing current events. The system supports that by organizing historical cases and showing where evidence exists or is missing.

The output is historical context, not prediction.

## Market Evidence Integration

The V2 market reaction pilot adds a market intelligence layer with observed historical returns and benchmark comparisons.

Business value:

- connects policy events to market evidence when data exists;
- documents proxy and benchmark choices;
- prevents unsupported market claims;
- separates available evidence from missing data.

## Executive Communication Support

The structured memo format helps analysts communicate with:

- portfolio managers;
- risk committees;
- strategy teams;
- corporate planning teams;
- portfolio reviewers.

The memo emphasizes bottom-line context, evidence, pathways, and caveats rather than forecasts or recommendations.
