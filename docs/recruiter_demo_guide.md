# Recruiter Demo Guide

This guide helps a recruiter, hiring manager, professor, or portfolio reviewer understand the project quickly.

## 5-Minute Review Path

### What to Read

1. [Project At A Glance](project_at_a_glance.md)
2. README sections:
   - Product Overview
   - Demo Workflow
   - Portfolio Value
   - Who Uses This Product?

### What to Ignore

- Source code implementation details.
- V2 schema and field-mapping documents.
- Long methodology documents.

### What the Reviewer Should Learn

- The project is a deterministic geopolitical intelligence workflow platform.
- It turns analyst questions into structured intelligence briefs.
- It demonstrates business analytics, risk analytics, workflow automation, and executive communication.
- It explicitly avoids forecasts, probabilities, and investment advice.

## 15-Minute Review Path

### What to Read

1. [Executive Demo Case](executive_demo_case.md)
2. [Scenario Intelligence Layer](scenario_intelligence_layer.md)
3. [Analyst Use Cases](analyst_use_cases.md)
4. [US-China Market Reaction Pilot Dataset](us_china_market_reaction_dataset.md)

### What to Ignore

- Internal coding templates.
- Full event dictionary.
- Detailed market-data codebook unless reviewing data architecture.

### What the Reviewer Should Learn

- How an analyst question moves through scenario family, analogues, market evidence, pathways, current relevance, and memo output.
- How V2 adds a small market-reaction evidence layer without becoming a forecasting tool.
- How the project supports analyst productivity and structured review.
